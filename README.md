# Net Movies

Net Movies is a mobile app built with Expo and React Native for discovering, searching, and saving movies.

The app uses TMDB for movie data and Supabase for authentication, user profiles, avatar storage, and bookmarks.

## What You Can Do

- Create an account and sign in
- Browse trending and popular movies
- Search movies by title
- View movie details
- Save favorites to bookmarks
- Edit your profile and avatar

## Stack

- Expo + React Native
- Expo Router
- Supabase
- TMDB API
